## 做什么

让更多人知道你的网站，并持续更新内容。

## SEO 优化提示词

> 📈 **推荐 Skill**：先安装 `create-prd` 中的文档写作能力，配合 AI 做 SEO 内容规划。

```bash
# 安装写作辅助 Skill（一键复制）
npx skills add mblode/agent-skills@docs-writing -g -y
```

然后把以下模板复制给 AI：

请帮我优化我的网站，让它在搜索引擎中更容易被找到。

网站 URL：[网址]
核心关键词：[你的目标搜索词]

请帮我：
1. 优化每个页面的标题和描述
2. 生成 sitemap.xml 文件
3. 建议做哪些内容来提升搜索排名

## 📋 更多提示词模板

**提示词 1：生成 sitemap.xml**

```text
我是一个完全不会编程的小白。

请帮我的网站生成一个 sitemap.xml 文件，方便搜索引擎收录。

我的网站网址：[你的网站 URL]
我的页面列表：
- 首页：[首页 URL]
- 关于：[关于页面 URL]
- 作品/服务：[页面 URL]
- 联系方式：[页面 URL]
- 博客/文章列表：[页面 URL]
- 其他页面：[页面 URL]

请帮我生成 sitemap.xml 文件的内容，要求：
1. 包含我上面列出的所有页面
2. 为每个页面设置合适的更新频率（daily / weekly / monthly）
3. 为每个页面设置优先级（首页 1.0，其他依次降低）
4. 包含 lastmod 日期

输出完整的 sitemap.xml 代码。
另外教我：这个文件应该放在网站的什么位置，以及如何提交给 Google Search Console。
```

**提示词 2：社交媒体推广文案**

```text
我是一个完全不会编程的小白，刚用 AI 做好了我的网站，想在社交媒体上宣传一下。

我的网站信息：
- 网站名称：[你的网站名称]
- 网站网址：[你的网站 URL]
- 网站类型：[个人博客 / 摄影作品集 / 电商 / 企业官网]
- 网站一句话介绍：[用一句话说明你的网站]
- 目标受众：[谁会来看这个网站]
- 我的社交媒体平台：[微信朋友圈 / 微博 / 小红书 / 知乎]

请帮我写 3 套不同风格的推广文案：

文案一：亲切朋友版
- 语气：像跟朋友聊天一样自然
- 适合发在：微信朋友圈

文案二：专业推荐版
- 语气：专业、可信
- 适合发在：知乎、LinkedIn

文案三：短小精悍版
- 语气：简洁有趣
- 适合发在：微博、小红书

每套文案包含：标题（或开头第一句）、正文（100 字以内）、标签/hashtag。
请用吸引人的方式写，让人想点开链接看看。

**任务**：优化你网站的 SEO，让 AI 生成 sitemap.xml，重新部署后搜索看看能不能搜到。
