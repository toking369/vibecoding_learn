## 推荐 Skill

> 🚀 **推荐 Skill**：`devops-deployment` 能帮你自动完成部署配置和 CI/CD 流水线。

```bash
# 安装 Skill（一键复制）
npx skills add yonatangross/orchestkit@devops-deployment -g -y
```

安装后在 Claude Code 中执行：

```bash
npx skills devops-deployment
```

## Vercel 部署步骤

1. 注册 Vercel（用 GitHub 登录）
2. 点击 "Add New → Project"
3. 选择你的 GitHub 仓库或直接拖拽文件夹
4. 等待自动部署完成
5. 访问生成的 URL

## 域名知识

| 概念 | 大白话 |
|------|--------|
| 域名 | 你网站的"门牌号"，比如 google.com |
| 免费子域名 | Vercel 给你的 my-site.vercel.app，够用了 |
| HTTPS | 网站地址栏的小锁🔒，Vercel 自动搞定 |
