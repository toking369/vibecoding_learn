## 做什么

写出吸引人的网站内容。

## 好的文案 vs 不好的文案

| ❌ 不好的文案 | ✅ 好的文案 |
|-------------|------------|
| "我们是专业的服务提供商" | "让 1000+ 客户放心的选择" |
| "我们的产品很好" | "3 个功能，帮你每天省下 2 小时" |
| "联系我们" | "聊聊你的想法？点击这里开始" |

## 网站文案提示词

> ✍️ **推荐 Skill**：先安装 `copywriting`，它会引导你完成从品牌调性到具体文案的全流程。

```bash
# 安装 Skill（一键复制）
npx skills add kostja94/marketing-skills@copywriting -g -y
```

安装后执行：

```bash
npx skills copywriting
```

如果不想用 Skill，也可以直接把以下模板复制给 AI：

请帮我的 [网站类型] 写文案。我需要：
1. 首页主标题和副标题（3 个方案供选择）
2. 关于我们页面文案（200 字以内）
3. 服务/产品介绍文案（每个功能一句话）

风格要求：亲切、可信、不浮夸

## 📋 更多提示词模板

**提示词 1：写品牌故事**

```text
我是一个完全不会编程的小白，想用 AI 帮我写网站的品牌故事。

我的品牌信息：
- 品牌/个人名称：[你的名字或品牌名]
- 我是做什么的：[一句话说明你的职业或业务]
- 我的故事：[讲讲你为什么做这件事，有什么特别的经历或理念]
- 我的价值观：[你坚持什么，如"诚信""创新""客户第一"]
- 目标用户：[谁会来找你]

请帮我写一篇"关于我"页面文案（300 字以内），要求：
1. 开头用一句有吸引力的话抓住读者
2. 讲清楚我做什么、为什么做、有什么不同
3. 语气亲切真实，不要套话
4. 结尾引导用户采取行动（如"来看看我的作品"或"联系我"）

请提供 2 个不同风格的版本：
版本 A：感性故事版——从个人经历出发，打动人心
版本 B：专业能力版——突出专业能力和成果，建立信任

每个版本请标注大概适合什么类型的网站。
```

**提示词 2：写产品/服务卖点**

```text
我是一个完全不会编程的小白，想用 AI 帮我写网站上的产品或服务介绍。

我的产品/服务信息：
- 产品/服务名称：[名称]
- 它是什么：[一句话说明]
- 核心功能：[列出 3-5 个主要功能或特点]
- 目标用户：[谁会买或用这个]
- 价格区间：[免费 / 99-199元 / 定制报价]
- 竞争对手：[你的竞品，没有就留空]

请帮我写出以下文案：

1. 产品标题（3 个方案）
   - 每个方案不同角度（功能导向、利益导向、情感导向）

2. 每个功能的一句话卖点
   - 不要说"有 XX 功能"，要说"能帮你解决 XX 问题"
   - 例如：不要说"24 小时客服"，要说"深夜遇到问题也有人帮你"

3. 价格/套餐介绍（简洁明了）

4. 常见问题回答（FAQ）3 条
   - 站在用户角度，写出他们最可能问的问题
   - 回答要简短、实在

风格要求：不说空话，每个卖点都要让读者觉得"这对我有用"。

**任务**：让 AI 为你的网站重新写首页主标题和副标题。选一个你最喜欢的版本，更新到网站上。
