## 做什么

确保网站正常工作，不出 bug。

## 测试提示词模板

> 🧪 **推荐 Skill**：先安装 `javascript-testing`，然后用 `/code-review` 命令让 AI 自动审查代码。

```bash
# 安装 Skill（一键复制）
npx skills add wshobson/agents@javascript-testing-patterns -g -y
```

安装后在 Claude Code 中执行：

```bash
/code-review
```

或者直接把以下模板复制给 AI：

请帮我审查以下 HTML/CSS/JS 代码，检查：
1. 有没有语法错误
2. 有没有链接打不开
3. 有没有移动端适配的问题
4. 有没有安全风险

## 快速测试法

打开浏览器 → 右键 → 检查（Inspect）→ Console（控制台）

如果看到红色错误信息，截图发给 AI 让它帮你解决。

## 📋 更多提示词模板

**提示词 1：浏览器兼容性检查**

```text
我是一个完全不会编程的小白，想检查我的网站在不同浏览器上是否正常。

我的网站用了以下技术：[HTML / CSS / JavaScript / Tailwind CSS / 其他]
我发现的问题：[描述你在某个浏览器看到的问题，如"在 Safari 上导航栏位置不对"]

请帮我分析以下兼容性问题：

1. 我的代码在哪些浏览器上可能有问题（Chrome / Firefox / Safari / Edge）
2. 常见的 CSS 兼容性问题（flexbox、grid、transform 等）在哪些浏览器上不支持
3. 我用的 JavaScript 语法（如箭头函数、模板字符串）在哪些旧浏览器上不支持
4. 针对发现的问题，给出修复方案

如果是 CSS 问题，请给出带浏览器前缀的替代写法（-webkit-、-moz- 等）。
如果是 JS 问题，请给出 polyfill 方案或替代语法。

请直接输出修改后的兼容代码。
```

**提示词 2：生成测试清单**

```text
我是一个完全不会编程的小白，想给我的网站做一个全面的测试清单。

我的网站信息：
- 网站类型：[你的网站类型]
- 包含页面：[列出所有页面]
- 主要功能：[联系表单 / 图片展示 / 用户登录 / 搜索等]

请帮我生成一个测试清单（checklist），按类别组织：

1. 功能测试
   - 每个页面的链接是否能正常跳转
   - 联系表单能否提交
   - [其他功能]

2. 显示测试
   - 在手机尺寸（375px）下显示是否正常
   - 在平板尺寸（768px）下显示是否正常
   - 在电脑尺寸（1440px）下显示是否正常

3. 内容测试
   - 是否有错别字或文案错误
   - 图片是否能正常加载
   - 是否有空链接或死链

4. 性能测试
   - 页面加载时间是否正常
   - 图片是否过大

请用表格形式输出清单，每项包含：测试项、测试方法、预期结果、结果（空着让我填）。

**任务**：让 AI 审查你的代码，生成检查清单，逐项检查你的网站。
