## 做什么

网站上线后，确保它稳定运行。

## 运维提示词模板

我的网站出问题了，请帮我诊断。
网址：[你的网站 URL]
问题描述：[描述你遇到的问题]

请告诉我最可能的原因是什么，以及我该怎么修复。

## 日常检查清单

- 网站能正常访问吗？
- 联系表单能正常提交吗？
- 链接都有效吗？
- SSL 证书过期了吗？（Vercel 自动更新）

## 📋 更多提示词模板

**提示词 1：网站性能体检**

```text
我是一个完全不会编程的小白，想检查我的网站性能好不好。

我的网站网址：[你的网站 URL]
我的网站用了：[纯 HTML / Astro / Next.js / 其他]

请帮我做一次全面的性能检查和建议：

1. 教我如何使用 Chrome 开发者工具检查性能（按 F12 → Lighthouse）
2. 常见的性能问题有哪些（图片太大、JS 太多、CSS 未压缩等）
3. 针对我的网站类型，给出 5 条具体的性能优化建议
4. 每条建议请给出具体的代码示例

我的网站目前主要问题是：[描述你感觉到的问题，如"打开很慢"或"滚动卡顿"]
```

**提示词 2：网站安全检查**

```text
我是一个完全不会编程的小白，想检查我的网站是否安全。

我的网站网址：[你的网站 URL]
我的网站技术栈：[纯静态 HTML / 有后端 / 有用户登录]
我用到的服务：[Vercel / Supabase / Formspree / 其他]

请帮我做一次安全检查，检查以下方面：

1. HTTPS 是否已开启（网站地址栏是否有小锁🔒）
2. 代码中是否不小心暴露了 API Key、密码或敏感信息
3. 联系表单是否有垃圾提交防护
4. 如果有用户登录，密码存储是否安全
5. 是否有基本的防 XSS（跨站脚本攻击）措施
6. 是否有防 SQL 注入措施（如果用了数据库）

对于每个检查项，请告诉我：
- 怎么检查（具体操作步骤）
- 如果发现问题，怎么修复（给出代码示例）

**任务**：让 AI 对你的已上线网站做一次全面的"健康检查"。按 AI 的建议改进至少 2 项。
