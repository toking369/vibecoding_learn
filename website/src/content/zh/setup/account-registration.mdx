## 注册 GitHub

1. 打开 [github.com](https://github.com)
2. 点击 "Sign up"
3. 输入用户名、邮箱、密码
4. 验证邮箱即可完成

## 注册 Vercel

1. 打开 [vercel.com](https://vercel.com)
2. 点击 "Sign Up" → "Continue with GitHub"
3. 授权后即可自动登录
4. 免费计划已经包含所有功能
