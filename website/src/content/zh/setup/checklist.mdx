## 你需要的东西

| 项目 | 推荐 | 费用 | 难度 |
|------|------|------|------|
| 电脑 | 任何 Windows / Mac 电脑 | 已有 | ★☆☆ |
| 浏览器 | Chrome（推荐）或 Edge | 免费 | ★☆☆ |
| AI 编程工具 | Claude Code 或 Cursor | 有免费版 | ★★☆ |
| 代码编辑器 | VS Code | 免费 | ★★☆ |
| 版本控制 | GitHub 账号 | 免费 | ★★☆ |
| 部署平台 | Vercel 或 Netlify 账号 | 免费 | ★★☆ |

## 验证环境

打开终端（Terminal），输入：
```bash
node --version   # 应该显示 v18 或更高
npm --version    # 应该显示 v9 或更高
```
