## 安装 VS Code

1. 访问 [code.visualstudio.com](https://code.visualstudio.com)
2. 点击下载 Windows/Mac 版本
3. 双击安装包，一直点"下一步"即可
4. 安装完成后打开 VS Code

## 安装 Node.js

1. 访问 [nodejs.org](https://nodejs.org)
2. 下载 LTS 版本（左侧按钮）
3. 双击安装，一直点"下一步"
4. 安装完成后，打开终端输入 `node --version` 验证
